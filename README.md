Manufacturing Quality Control & Defect Analytics Database

Build a manufacturing database tracking production metrics and defects. 
Must include defect rate monitoring triggers and quality trend windows.

Project Scope:
The scope of this project is to build a database that monitors for maufacturing quality and defects for production means. Along with using the data given to see broader trends and predict future quality and defects in the production process. The database will only include date and time for each measured item, as well as the quality for each item measured, and any other information important to predict future quality trends. 

Users:
The intended users for this database are Quality Control Analyst, Manufacturing Engineers, and Operations Managers.

Data Source:
To acheve this project's goal data from kaggle using the following link, https://www.kaggle.com/datasets/edumagalhaes/quality-prediction-in-a-mining-process/data , will be used as the data for the database. With this data specifically containing information about iron ore, the date and time of each measured sample and quality/purity of the ore, as well as other variables during post process of the ore that affects quality, and the final ore purity after post processing.
